# Homebrew Tap for agentviewer

This is the official Homebrew tap for [agentviewer](https://github.com/pengelbrecht/agentviewer).

## Installation

```bash
brew install pengelbrecht/agentviewer/agentviewer
```

## Updating

```bash
brew update
brew upgrade agentviewer
```

## Uninstalling

```bash
brew uninstall agentviewer
brew untap pengelbrecht/agentviewer
```
